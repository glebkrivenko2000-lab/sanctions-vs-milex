# Data Sources and Reproduction Instructions

Due to GitHub's file size limits and data redistribution licenses, the raw datasets are not included directly in this repository. To fully reproduce the analysis, please download the required datasets from their official sources and place them into the `data/raw/` directory.

The custom country code crosswalk file (`perehod.xlsx`) is provided in the repository to map various historical and database-specific codes (e.g., COW, UNDP) to standard ISO-3 format.

## Required Datasets

### 1. SIPRI Military Expenditure Database
* **Description:** Absolute military spending and military expenditure as a share of GDP.
* **Source:** [Stockholm International Peace Research Institute (SIPRI)](https://www.sipri.org/databases/milex)
* **Action:** Download the "SIPRI Military Expenditure Database" Excel file.
* **Expected Filename:** `SIPRI-Milex-data-1948-2023_0.xlsx`

### 2. Global Sanctions Data Base (GSDB)
* **Description:** Dyadic sanction data containing various types of sanctions and their intensities.
* **Source:** [GSDB Official Website](https://www.globalsanctionsdatabase.com/) (Requires a short registration for academic access).
* **Action:** Request and download the **Dyadic version** of Release 4.
* **Expected Filename:** `GSDB_V4_dyadic.dta`

### 3. Correlates of War (COW) - Bilateral Trade
* **Description:** Bilateral trade flows between state dyads.
* **Source:** [Correlates of War Project (Version 4.0)](https://correlatesofwar.org/data-sets/bilateral-trade/)
* **Action:** Download the Dyadic Trade dataset ZIP archive and extract the CSV.
* **Expected Filename:** `Dyadic_COW_4.0.csv`

### 4. Correlates of War (COW) - Militarized Interstate Disputes (MID)
* **Description:** Data on armed conflicts and wars between states.
* **Source:** [Correlates of War Project (MID v4.3 or v5.0)](https://correlatesofwar.org/data-sets/mids/)
* **Action:** Download the Dyadic MID dataset.
* **Expected Filename:** `dyadic_mid_4.03.csv` *(Note: If using a newer version, rename the file or adjust the path in `01_data_preparation.py`)*.

### 5. CEPII GeoDist
* **Description:** Exogenous geographic and historical bilateral frictions (distance, contiguity, language, colonial links) used for the gravity model.
* **Source:** [CEPII GeoDist Database](http://www.cepii.fr/CEPII/en/bdd_modele/presentation.asp?id=6)
* **Action:** Download the Stata (`.dta`) version of the dataset.
* **Expected Filename:** `dist_cepii.dta`

### 6. Polity V Project
* **Description:** Institutional characteristics and democratization scores.
* **Source:** [Center for Systemic Peace](http://www.systemicpeace.org/inscrdata.html)
* **Action:** Download the "Polity5 Annual Time-Series, 1946-2018" Excel file.
* **Expected Filename:** `polity5.xls`

### 7. UN General Assembly Voting Data
* **Description:** Dyadic political distance calculated from UNGA voting records.
* **Source:** [UN Digital Library](https://digitallibrary.un.org/) / Harvard Dataverse (Voeten et al.).
* **Action:** Download the dyadic UNGA voting dataset up to 2025.
* **Expected Filename:** `2025_9_19_ga_voting.csv`

## Directory Structure

Before running the scripts, ensure your `data/` folder looks exactly like this:

```text
Economic-Sanctions-Military-Spending/
├── data/
│   ├── DATA_README.md
│   ├── raw/
│   │   ├── SIPRI-Milex-data-1948-2023_0.xlsx
│   │   ├── GSDB_V4_dyadic.dta
│   │   ├── Dyadic_COW_4.0.csv
│   │   ├── dyadic_mid_4.03.csv
│   │   ├── dist_cepii.dta
│   │   ├── polity5.xls
│   │   ├── 2025_9_19_ga_voting.csv
│   │   └── perehod.xlsx               <-- (Included in this repo)
│   └── cleaned/                       <-- (Will be generated by the code)
```

Once the files are in place, you can execute `01_data_preparation.py` to initiate the data pipeline.
